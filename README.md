## Installation

-   Install Make

-   Start database

```bash
$ make db-start

$ go get
```

## Run

1. Basic

```bash

$ go run main.go

```

2. Live reload

Install [air](https://github.com/air-verse/air)

```bash
$ make install-air

# run live reload
$ make run-dev

# or

$ make all
```
